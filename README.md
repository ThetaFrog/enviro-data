# enviro-data

_**Brief**_:

Enviro-data is a personal data analysis project focused on environmental science and policy. The project uses a two-pronged approach: exploring high-level environmental concepts downward into the data, while simultaneously aggregating raw data to uncover bottom-up insights. The repository is organized into distinct, topic-focused modules containing data pipelines, visualizations, and written analyses.

_**Lines of Inquiry**_:
* **Explore empirical data to broaden understanding of natural phenomena**: things like storm data, land use by geography etc. are important to know in themselves, but also act as metrics which can be applied to higher-level analysis.
* **Phenomenological synthesis:** Combined insights from multiple data sources can be used to build deeper understandings of larger, more complex phenomena.
* **Policy and Trend Analysis:** longitudinal data from multiple sources can be used to gain an understanding of how company or governmental policy is helping or hurting the environment in the long term.

**_Purposes_**:
* **Technical (Personal)**: Learn about processing, cleaning, presenting and analysing data using Python.
* **Impact**: Present otherwise difficult to interpret data in clear and interactive ways to raise awareness for environmental issues
* **Professional**: Build a public portfolio demonstrating competence in data engineering, statistical analysis, and data storytelling.
